# IEC60870-test
Это пример практической работы помогающей погрузиться в особенности протокола МЭК 60870-5 104 основанный на использовании виртуальных Сервер, Клиент. 
В работе разобран способ запуска Сервера и Клиента из примеров публичной библиотеки lib60870-C (https://github.com/mz-automation/lib60870/blob/master/README.md) 
Так же выполнено отслеживание пакетов данных передаваемых между Виртуальными Сервер-Клиент которые передают данные по протоколу МЭК-104.
Подробную инструкцию и ход выполнения с результатами работы можно посмотреть в прилагаемом pdf документе.


[English report]
Practical analysis of IEC 60870-5-104 protocol traffic between virtual client and server using lib60870-C and Wireshark.

This is a hands-on lab designed to help dive into the specifics of the IEC 60870-5-104 protocol using a virtual Server-Client setup.

The project demonstrates how to run the Server and Client examples from the open-source lib60870-C library
(https://github.com/mz-automation/lib60870).

It also includes real-time packet capture and analysis of IEC-104 traffic exchanged between the virtual Server and Client.

For a detailed step-by-step guide and complete results, refer to the attached PDF document.
